# Guard Alert System

The Guard Alert System is a real-time drowsiness detection application that uses computer vision to monitor eye activity through a webcam. It alerts the user with an alarm when signs of drowsiness or sleep are detected, helping improve safety for security guards and night-shift personnel.

# Features

Real-time face detection using Dlib

Eye blink analysis with 68 facial landmarks

Detects Active, Drowsy, and Sleeping states

Audio alarm on prolonged eye closure

Live webcam feed with facial landmarks

# Technologies Used

Python

OpenCV

Dlib

NumPy

Imutils

Pygame

# Requirements

shape_predictor_68_face_landmarks.dat

alarm.wav

Webcam-enabled system

# How to Run
1. Ensure Python is installed on your system.
2. Open the terminal in the project directory.
3. Run the following command:
python guard_alert_system.py


Press ESC to exit the application.

# Use Case

This system helps prevent accidents caused by fatigue by actively monitoring alertness and providing timely alerts.
